# 도구 누락 안내 메시지 — `/kdesigner:디자인초기설정` §0

> `/kdesigner:디자인초기설정` §0 도구 사전 확인에서 필수·선택 도구가 빠진 경우 *그대로 응답에 박을* 한국어 안내 메시지. 분기 로직은 슬래시 본체 §0에 유지, 메시지 본문만 이 문서에 분리.
>
> 슬래시는 `${CLAUDE_PLUGIN_DATA}` 같은 환경변수보다 *문서 경로 직접 인용*이 안정 — 본체에서 "아래 메시지를 그대로 노출" 식으로 참조.

---

## 필수 도구 누락 (흐름 정지)

영어 에러 한 줄도 노출 X. *해당하는 항목만* 묶어 한 응답에 함께 노출.

### Homebrew 미설치 (macOS 한정, 권장)

```
**도구 설치 도우미**(`Homebrew`)가 아직 없어요 — 한 줄로 다른 도구들을 깔 수 있게 해주는 macOS용 도우미라, 깔아두시면 아래 도구들 설치가 훨씬 편해져요. 공식 사이트(`https://brew.sh`) 첫 화면에 *복사해서 터미널에 붙여넣을* 한 줄 명령이 있어요 — 그대로 실행하시면 돼요(약 5–10분, 중간에 비밀번호 한 번 물어요).

brew 없이도 진행은 가능해요(아래 fallback 경로). 깔지 말지는 자유롭게 결정해주세요.
```

### Node.js 미설치

**macOS + brew 있음**:
```
**화면 엔진**(`Node.js`)이 깔려 있지 않아요 — 디자이너가 만든 화면을 띄우는 데 꼭 필요한 도구예요. 터미널에 `brew install node`를 입력하시면 약 1분 안에 깔려요. 끝나면 *지금 열려 있는 터미널·대화창을 닫고 다시 열어주세요* — 새로 깐 도구가 그제서야 인식돼요.
```

**그 외 (brew 없음 / Windows / Linux)**:
```
**화면 엔진**(`Node.js`)이 깔려 있지 않아요 — 디자이너가 만든 화면을 띄우는 데 꼭 필요한 도구예요. 공식 사이트(`https://nodejs.org`)에서 *LTS* 버전 설치 파일을 받아 실행하시면 돼요. 설치가 끝나면 *지금 열려 있는 터미널·대화창을 닫고 다시 열어주세요* — 새로 깐 도구가 그제서야 인식돼요.
```

### git 미설치

**macOS + brew 있음**:
```
**작업 시점 저장 도구**(`git`)가 깔려 있지 않아요 — "방금 작업 그대로 다시 돌아가기" 같은 안전망에 꼭 필요해요. 터미널에 `brew install git`를 입력하시면 1–2분 안에 끝나요.
```

**macOS + brew 없음**:
```
**작업 시점 저장 도구**(`git`)가 깔려 있지 않아요. 터미널에 `xcode-select --install`을 한 번 입력하시면 자동 설치 창이 떠요(약 5분).
```

**Windows**:
```
**작업 시점 저장 도구**(`git`)가 깔려 있지 않아요. 공식 사이트(`https://git-scm.com`) 안내를 따라가시면 돼요.
```

### git 사용자 정보 미설정 (`user.name` 또는 `user.email` 비어있음)

```
**누가 만든 작업인지 표시할 정보**(이름·이메일)가 아직 없어요. 저장 시점(`commit`)마다 한 줄로 들어가요 — 한 번만 알려주시면 다시 묻지 않아요.
```

처리 절차: `AskUserQuestion`으로 이름·이메일 묻기. 받은 값으로 `git config --global user.name "..."` / `git config --global user.email "..."` 실행. 사용자가 거부하면 안내 한 줄 후 흐름 중단 — 강제 X.

### 흐름 정지 시 응답 끝 안내

필수 도구 누락으로 흐름을 멈춘 경우 응답 마지막에:

```
위 도구들이 다 준비되면 **`/kdesigner:디자인초기설정`을 한 번 더 불러주세요** — 같은 자리에서 이어서 시작해요.
```

다음 행동 1개 제안(글로벌 §7) 자리는 이 안내가 대체.

## 선택 도구 누락 (안내만, 흐름 진행)

### gh 미설치 또는 `gh auth status` 실패

**macOS + brew 있음**:
```
**팀 공유 공간 연결**(`GitHub` + `gh`)은 아직 준비 안 됐어요. 지금 당장은 없어도 디자인 작업은 시작할 수 있고, 나중에 *개발자한테 넘기기* 단계에서 필요해져요. 미리 깔아두시려면 `brew install gh` 한 줄이면 돼요.
```

**그 외**:
```
**팀 공유 공간 연결**(`GitHub` + `gh`)은 아직 준비 안 됐어요. 지금 당장은 없어도 디자인 작업은 시작할 수 있고, 나중에 *개발자한테 넘기기* 단계에서 필요해져요. 미리 깔아두시려면 공식 사이트(`https://cli.github.com`) 안내 따라 설치하시면 돼요.
```

### gh는 깔려 있는데 로그인만 안 된 경우

```
**팀 공유 공간**(`GitHub`) 도구는 깔려 있는데 로그인은 아직이에요. 나중에 *개발자한테 넘기기* 단계에서 필요해질 때 다시 안내드릴게요.
```
